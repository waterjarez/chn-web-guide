# chn-web-guide

## 项目简介

`chn-web-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用 HTML 页面集合的存储与展示空间。每个页面均为独立内容，可单独访问或浏览。

## 目录结构

```
.
├── README.md          # 本说明文件
├── index.html         # 仓库首页/导航页（如有）
├── pages/             # 独立 HTML 页面存放目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 公共资源（样式、图片、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有独立 HTML 页面文件。
- `assets/`：存放页面共享的资源文件，如样式表、JavaScript 脚本和图片。

## 页面归档说明

本仓库收录的 HTML 页面均为独立制作或整理，内容涵盖多种类型。每个页面均经过基本测试，确保在主流浏览器中可正常显示。页面之间无直接关联，可单独使用或引用。

**注意**：页面内容可能随时间更新，请以仓库最新版本为准。

## 使用方法

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看对应页面。
3. 若需部署到 Web 服务器，将整个仓库目录上传至服务器根目录或子目录即可。
